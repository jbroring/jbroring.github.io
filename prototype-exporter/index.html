Install the plugin in Figma

How does it work?

  1. Use Figma to design and create a prototype
  2. Go to the page with your prototype & select the Starting Frame
  3. Open Prototype Exporter
  4. Click "Download prototype" to download a .zip file
  5. Unzip & open index.html in your (users) favorite browser!

What (Figma) features are supported and is not supported?

  • Interaction: On Click → Navigate to Frame
  • Layers inside groups
  • Optional 2x retina image export
Not (yet) supported
  • Fixed layers are currently ignored (only whole frames are being exported)
  • Interaction: While Hovering
  • Animations are currently ignored (everything is an "Instant" animation)
  • Preserve scroll position is ignored (click will navigate and scroll to top)
  • Overlays do not function yet (you can them on top of a frame, for now)
  • Overflow scrolling (Use longer frames instead, for now)
  • Background color for the prototype that you may have set

Frequently asked questions

Does Prototype Exporter change something in my Figma design?
No, it does not! The plugin only reads your file and uses that data to create the exported prototype.
